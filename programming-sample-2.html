Unreal Editor for Fortnite

Project Goal

This project showcases my role as Technical Director for several key Unreal Editor for Fortnite (UEFN) templates. My responsibility was to oversee the technical implementation, ensure best practices, and guide the development of these foundational examples for the UEFN community, demonstrating the power of the Verse programming language.

Tools/Languages: Unreal Editor for Fortnite (UEFN), Verse Programming Language, MetaHuman, Niagara, and Sequencer.

Key Contributions

As Technical Director, I was responsible for the technical design and execution of templates that serve as the starting point for thousands of creators. My work involved architecture, code review, and establishing best practices for the Verse programming language.

  • Verse Starter Template: Directed the technical development of this template, designed to introduce creators to the power of the Verse programming language for creating custom gameplay logic in UEFN.
  • Talisman Metahuman Template: Oversaw the integration of MetaHuman assets into UEFN, establishing a pipeline and best practices for using high-fidelity characters in Fortnite experiences.
  • Talisman Environment Template: Guided the team in leveraging high-quality assets from the Talisman demo to showcase advanced environment art and cinematic capabilities within UEFN.
