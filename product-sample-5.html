Epic Online Services

Training Content Developer

I developed tools that enabled me and my team to autogenerate documentation from the source code that was being developed by Team Online for the release of Epic Online Services.

Key Contributions

  • Visual Studio Extension: Developed a Visual Studio Extension that templatized a Diataxis approach to writing documentation. This extension was used internally by engineering and technical writing teams to automatically generate the metadata, structured markdown syntax, and template language for conceptual overviews, how-to guides, quick start guides, and reference documents, improving the quality and quantity of documentation being generated for Unreal Engine.
  • Unreal Engine Plugin: Developed an Unreal Engine plugin for our tutorial system, enabling engineers to wrap functional engine and gameplay code into snippets that APIDocTool could ingest for the C++ API Reference Manual.
  • APIDocTool: Debugged Linux, Android, and iOS API calls into APIDocTool, building the hooks for Unreal's reflection system to detect new code being written for upcoming releases of UE4, also maintained and extended the Code Snippet and Statistics Subsystems as the engine grew over time.
  • FPS Sample: Migrated legacy components from Unreal Tournament into Unreal Engine's First Person Shooter sample, writing one of the most downloaded and widely used tutorials on Epic's documentation site.
